This assignment is about train booking system. It allows

Passengers to:
1. register as member
2. login
3. book seats(select origin, destination, no of pax, date, time-specific-train, seats)
4. make payment

Admin to:
1. sign up
2. login

Languages and Tools used:
1. C#
2. Xaml
3. Visual studio
4. Firebase database
5. .NetCore for UWP
